# Ethiopian Time Converter

A Flutter mobile application that converts international time (Gregorian) into the Ethiopian calendar and Ethiopian clock system.

## Features
- Select city/timezone from a global list.
- Fetch real-time local data via WorldTimeAPI.
- Convert Gregorian date to Ethiopian date (13 months).
- Convert 24-hour clock to Ethiopian 12-hour clock (Day/Night system).
- Side-by-side comparison of results.

## Getting Started
1. Clone the repository.
2. Run `flutter pub get`.
3. Run `flutter run`.
